The HTML page structure
------------------------------
<html>
	<head>
		<title>    
		</title>
	</head>
	<body>
		
	</body>
</html>

<Title> : - this tag use to give name to Tab.
<input> :- For user imput 
Exaple of Text box
-------------------
<html>
	<head>
		<title>Home</title>
	</head>
	<body>
		<table>  
			<tr>
				<td><span>Enter User Name</span></td>
				<td><input type="text"/></td>
			</tr>
			<tr>
				<td><span>Enter Password</span></td>
				<td><input type="text"/></td>
			</tr>
		</table>
	</body>
</html>
